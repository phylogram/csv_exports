Basic program flow of collecting and storing data:
==================================================

1) Input default:
-- time-start = last request
-- time-end = now
-- topic = all
2) if time-start = 'last-request':
	get last request time from database
   if time-end-get:
	get time now
2) File-System.folder(year, month):
	check if folders exist, else create
3) foreach topic get new Class database-connection-child:
	File-System.file(topic):
		check if topic exists else create
		return file & temp file copy [rollback]
	foreach fetch-row:
		clean row
		append row to temp file copy
		each x.th request: clean up
		
		on error: Rollback: remove temp file, log to db

	- Fielsystem.update(): remove file, rename tempfile to file
	- log to db
